ReSpace
=========

Respace is a web based IDE for guided and interactive programming assignments.

## Prerequisites
1. NodeJS (> 6)
2. Yarn

## Install

    yarn install

## Dev + Hot Reload + Test Watcher

    yarn run dev
    # Then navigate to localhost:8000.
    
    # Run Test Watchers
    ./node_modules/bin/jest --watch

## Bundling

    yarn run bundle

## Testing+Coverage

    ./node_modules/bin/jest --coverage
